◆ Step8_API設計の要点

 1-1.クライアントからのリクエストをJSONに変換して送信する

 1-2.リクエストに対するレスポンスをJSON形式で返すようにする
  方法：
  　・既存処理：server.get("/",(req,res) =>
               {
                    res.send("レスポンスを文字列で返します。");
               });
    ・JSON   ：server.get("/",(req,res) =>
               {
                    res.json(
                    {
                        message:"レスポンスをJSON形式で返します。"
                    });
               });
